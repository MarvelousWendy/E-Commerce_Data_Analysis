# E-Commerce Data Analysis

This project explores an **e-commerce transaction dataset from Kaggle** to uncover patterns in customer behavior, product demand, and sales performance using data analysis techniques.

The analysis demonstrates practical skills in **data cleaning, exploratory data analysis (EDA), statistical analysis, and data visualization** using Python.

---

## Project Objectives

- Analyze customer demographics and purchasing behavior
- Identify sales trends across age groups and product categories
- Examine time-series patterns in transactions and revenue
- Segment and rank customers based on purchasing activity
- Evaluate payment method usage and transaction distribution
- Explore correlations between key business variables

---

## Key Analyses

### Data Overview & Cleaning
- Dataset structure inspection
- Summary statistics
- Missing value checks

### Demographic Analysis
- Age distribution of customers
- Purchasing behavior by demographic groups

### Product & Sales Trends
- Most popular product categories
- Revenue contributions by product type

### Time Series Analysis
- Transaction patterns over time
- Revenue trends across time periods

### Customer Segmentation
- Ranking customers by purchasing behavior
- Identifying high-value customers

### Payment Method Analysis
- Distribution of payment methods
- Relationship between payment types and sales

### Correlation Analysis
- Relationships between customer attributes, product purchases, and sales metrics
